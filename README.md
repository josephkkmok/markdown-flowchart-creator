1. Open `SKILL.md`.

## Example prompts

- Create a flowchart in markdown for user signup.
- Generate a flow diagram as markdown for payment retries.
- Make an ASCII flowchart for our CI/CD workflow.
- Document the flow of request routing in markdown.

## Output

This guide produces a documentation-style Markdown file with:

- A high-level ASCII diagram of the flow
- Explanations for each path and decision
- Real-world scenarios
- Configuration notes
- Benefits and trade-offs
- Maintenance commands (when relevant)

## Notes

Keep `SKILL.md` (uppercase) to avoid case-sensitivity issues on some filesystems.
